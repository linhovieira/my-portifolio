# 💼 Paulo Vieira | Professional Portfolio

> Um portfólio moderno desenvolvido com **Next.js**, **React** e **TypeScript**, projetado para representar minha identidade profissional como Engenheiro de Software e demonstrar excelência em engenharia, arquitetura e experiência do usuário.

---

# 📖 Sobre o Projeto

Este repositório contém o código-fonte do meu portfólio profissional.

Mais do que um site pessoal, este projeto foi concebido como um produto digital, aplicando princípio de arquitetura, Design System, componentização, acessibilidade e boas práticas de engenharia de software.

O objetivo é demonstrar, através do próprio código, a mesma qualidade esperada em aplicações corporativas modernas.

---

# 🎯 Objetivos

* Apresentar minha trajetória profissional.
* Centralizar informações sobre minha carreira.
* Demonstrar conhecimento em Engenharia de Software.
* Servir como destino principal do QR code presente no currículo.
* Disponibilizar uma versão online do currículo.
* Permitir futura geração de currículo em PDF.
* Evoluir continuamente conforme minha carreira.
